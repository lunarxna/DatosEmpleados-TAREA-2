# DatosEmpleados
Ejercicio sobre el manejo de archivos de texto y validaciones

Descripción:
Este es un programa realizado con Windows Forms que permite ingresar los datos de empleados com Id, Nombre, Apellidos, Telefono, Género, etc. El programa valida si te falto poner algo o pusiste algo por error y lanza un mensaje de alerta, te permite guardar los datos como si fuera guardando un archivo en Wird y los datos se guardan en forma de tabla, permite de igual forma abrir archivos ya guardados, refrescar los datos y salir del programa al terminar.

Herramientas utilizadas:
- Lenguaje: C#
- Framework: .NET Framework
- IDE: Visual Studio 2022
